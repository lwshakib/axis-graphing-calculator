## How Can I Contribute?

### Reporting Bugs
If you find a bug, please create an issue on GitHub. Include as much detail as possible, such as:
- A clear description of the bug.
- Steps to reproduce it.
- Expected vs. actual behavior.

### Suggesting Features
We love new ideas! If you have a suggestion:
- Check the issues to see if it's already been suggested.
- If not, open an issue labeled "feature request".

### Pull Requests
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/your-feature-name`).
3. Make your changes.
4. Commit your changes with a clear message.
5. Push to your fork and submit a Pull Request.

## Style Guidelines

- Please use consistent naming conventions.
- Ensure your code is well-commented where necessary.
- Follow the existing project structure.

## Code of Conduct

Participation in this project is governed by our [Code of Conduct](CODE_OF_CONDUCT.md). By contributing, you are expected to uphold this code.

Thank you for making Axis Graphing Calculator better!
